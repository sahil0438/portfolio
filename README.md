## Setup & Usage

### Option 1 — Open directly
Just open `index.html` in any modern browser. No server needed for basic viewing.

### Option 2 — With Sass (for editing styles)

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Build CSS from SCSS once**
   ```bash
   npm run sass:build
   ```

3. **Watch for SCSS changes during development**
   ```bash
   npm run sass:watch
   ```

### Option 3 — Deploy on Netlify
This site includes `netlify` on the contact form, so deploying on [Netlify](https://netlify.com) will automatically enable form submissions with no backend required.
